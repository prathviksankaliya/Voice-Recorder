# Voice Recorder Examples

Simple, easy-to-understand examples for beginners.

### Example 1: Basic Recording Controls
**File**: `lib/examples/example1_basic_controls.dart`

Learn the basics:
- Initialize VoiceRecorder
- Start/Pause/Resume/Stop recording
- Track recording state
- Display duration
- Delete recording

### Example 2: Customization
**File**: `lib/examples/example2_customization.dart`

Learn to customize:
- Audio quality presets (low, medium, high, voice)
- Storage locations (temp, visible, custom)
- Pass configuration to recorder

### Example 3: Wave Visualization
**File**: `lib/examples/example3_wave_visualization.dart`

Learn wave features:
- Use AudioWaveWidget
- Apply wave presets (minimal, standard, modern)
- Customize colors and styles
- Add decorations

### Example 4: Complete App
**File**: `lib/examples/example4_complete_app.dart`

Production-ready example with:
- All recording controls
- Wave visualization
- Duration tracking
- Error handling
- Interruption handling
- File metadata
